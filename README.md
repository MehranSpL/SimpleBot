# How to install

`Clone The Repo`

`Open Your Command Prompt Or Terminal`

`Run - npm install discord.js@12.0.1`

`After Installed Go Inside Config.json file and replace Your Bot token with (yourbottoken)`

`Save Files`

`Use - node index.js`

`Enjoy!`
